# 네이버 블로그 SEO 가이드

## 개요
본 가이드는 네이버 블로그의 검색 엔진 최적화(SEO)를 위한 표준 작성 규칙입니다.

---

## 1. 제목 규칙

### 1.1 제목 길이
- **20자 이내 작성** (필수)
- 띄어쓰기 포함 계산
- 메인 키워드는 제목 앞쪽에 배치

### 1.2 제목 작성 팁
```
✓ 산행 초보자 필수 준비물 10가지
✓ 북한산 가는 길 완벽 가이드
✗ 산에 가려고 하는 사람들이 꼭 알아야 할 중요한 준비물 리스트
✗ 산행할 때 챙겨야 할 물건들을 알려드릴게요
```

- 숫자나 기호 포함하면 클릭률 증가
- 시간, 팁, 가이드 등 실용적 단어 사용
- 키워드 중복 피하기

### 1.3 키워드 배치
- 메인 키워드를 제목의 **앞쪽(처음 10자 이내)** 에 배치
- 검색 알고리즘이 앞부분을 더 중요하게 평가

---

## 2. 본문 구조

### 2.1 필수 구성
본문은 다음 **6단계 순서**로 작성합니다:

```
1. 주제 요약 (2-3줄)
   ↓
2. 이미지 (1개 필수)
   ↓
3. 본문1 (첫 번째 핵심 내용)
   ↓
4. 본문2 (두 번째 핵심 내용)
   ↓
5. 본문3 (세 번째 핵심 내용)
   ↓
6. 마무리 (결론 및 행동 유도)
```

### 2.2 각 섹션 상세 설명

#### 1단계: 주제 요약
- 본문 맨 앞에 배치
- 2-3줄, 50-100자 분량
- 글의 주제와 핵심을 한눈에 파악 가능하게 작성
- 메인 키워드 포함 (첫 1단락 이내)

**예시:**
```
이 글에서는 산행 초보자를 위한 10가지 필수 준비물을 소개합니다.
안전하고 즐거운 산행을 위해 꼭 챙겨야 할 물건들을 단계별로 알려드리겠습니다.
```

#### 2단계: 이미지 (1개)
- **1개 필수, 2개 이상 권장하지 않음**
- 고화질 이미지 (최소 800x600px 이상)
- 본문과 관련 있는 이미지
- Alt 텍스트 설정 (SEO에 도움)

#### 3-5단계: 본문1, 본문2, 본문3
- 각 섹션마다 150-200자 (최소 100자 이상)
- 소제목 사용으로 구조 명확하게
- 줄바꿈으로 단락 구분
- 관련 부제목 사용 예시:
  ```
  ■ 본문1 : 제목 또는 핵심 키워드
  설명 내용...
  
  ■ 본문2 : 제목 또는 핵심 키워드
  설명 내용...
  ```

#### 6단계: 마무리
- 결론 또는 요약 (2-3줄)
- 독자 행동 유도 (댓글, 공감, 구독 권장)
- 관련 다른 글로 유도 가능

---

## 3. 키워드 배치 규칙

### 3.1 주요 키워드 배치 위치
1. **제목** (가장 중요)
2. **주제 요약** (첫 단락)
3. **소제목** (본문1, 2, 3)
4. **본문 중간** (자연스럽게 2-3회)

### 3.2 키워드 최적화
```
✓ 자연스러운 키워드 사용
✗ 키워드 반복이 과도함 (같은 단어 5회 이상)
✗ 숨겨진 텍스트에 키워드 삽입
```

- 주요 키워드: 3-5회 자연스럽게 배치
- 연관 키워드: 함께 사용 (예: "산행" + "등산" + "등산복")
- 동의어 활용으로 자연스럽게 확장

### 3.3 연관 검색어 활용
- 네이버 검색창 자동완성 활용
- 본문에서 관련 검색어 자연스럽게 포함
- 스팸성 키워드 삽입 피하기

---

## 4. 이미지 규칙

### 4.1 이미지 개수 및 크기
- **본문당 1개 권장** (정해진 규칙)
- 최소 해상도: 800x600px
- 권장 해상도: 1200x800px 이상
- JPG 또는 PNG 형식

### 4.2 이미지 최적화
```
✓ 본문과 관련성 높은 이미지
✓ 고화질, 깨끗한 이미지
✓ 캡션이나 설명 추가
✗ 저화질 이미지
✗ 관련 없는 이미지
```

### 4.3 이미지 배치 위치
- 주제 요약 다음, 본문1 이전에 배치
- 이미지 아래에 간단한 설명 추가
- Alt 텍스트 설정: 이미지 설명 (예: "산행 준비물 종류")

---

## 5. 본문 길이 가이드

### 5.1 최소 본문 길이
- **전체 본문 600자 이상** (권장)
- 각 섹션(본문1, 2, 3) 최소 100자 이상
- 너무 짧은 글은 검색 순위에 불리

### 5.2 본문 구성 분배
```
주제 요약: 50-100자
본문1: 150-200자
본문2: 150-200자
본문3: 150-200자
마무리: 50-100자
─────────────
합계: 600-800자
```

---

## 6. 태그 및 카테고리

### 6.1 태그 규칙
- **3-5개 태그** 설정 (너무 많으면 스팸으로 판단)
- 본문의 핵심 키워드 기반
- 포괄적인 태그보다 구체적인 태그 우선

**예시:**
```
✓ 산행, 등산 준비물, 북한산, 초보자 가이드, 등산복
✗ 여행, 운동, 취미, 일상, 추천
```

### 6.2 카테고리
- 블로그 카테고리를 일관되게 유지
- 같은 주제끼리 같은 카테고리 분류
- 깊이: 2-3단계 (너무 깊으면 접근성 낮음)

---

## 7. 내부링크 및 외부 기타 요소

### 7.1 내부링크
- 관련 다른 글 1-2개 링크
- 글 마무리 부분이나 본문 중간에 자연스럽게 배치
- "관련글 보기", "다른 글 읽기" 등으로 표현

### 7.2 댓글 및 상호작용
- 댓글 달린 글이 검색 순위에 유리
- 글 마무리에서 독자 의견 유도
- 댓글에 성실하게 답변

---

## 8. 하지 말것 (금지사항)

### 8.1 콘텐츠 관련
```
✗ 타 블로그 글 무단 복사/베껴쓰기
✗ AI 생성 콘텐츠 (저품질, 무의미한 내용)
✗ 같은 키워드 과도하게 반복 (5회 이상)
✗ 숨겨진 텍스트나 링크 삽입
✗ 너무 짧은 본문 (200자 미만)
✗ 본문과 무관한 이미지 배치
```

### 8.2 SEO 스팸 행위
```
✗ 클로킹 (검색 봇과 사용자에게 다른 콘텐츠 제공)
✗ 키워드 스터핑 (무의미한 키워드 반복)
✗ 자동 댓글 생성 프로그램 사용
✗ 과도한 자기 사이트 링크 삽입
✗ 링크 팜 (검색순위 조작용 링크 모음)
```

### 8.3 이미지 관련
```
✗ 저화질 이미지 (640x480px 이하)
✗ 저작권 침해 이미지 (무단 사용)
✗ 과도한 이미지 개수 (5개 이상)
✗ 압축되지 않은 무거운 파일
```

### 8.4 형식 관련
```
✗ 계속 변경되는 제목 (한 달에 3회 이상)
✗ 같은 내용의 중복 글
✗ 이전 글의 단순 수정 재발행
✗ 오류 많은 맞춤법과 띄어쓰기
```

## 9. 체크리스트 (글 발행 전)

글을 완성하기 전에 다음을 확인하세요:

### 제목 및 키워드
- [ ] 제목이 20자 이내인가?
- [ ] 메인 키워드가 제목에 포함되어 있는가?
- [ ] 제목이 명확하고 매력적인가?
- [ ] 태그는 3-5개로 설정되었는가?

### 본문 구조
- [ ] 주제 요약이 명확한가? (50-100자)
- [ ] 이미지가 1개 정확히 포함되었는가?
- [ ] 본문1, 2, 3이 각각 100자 이상인가?
- [ ] 마무리 문장이 있는가?
- [ ] 전체 본문이 600자 이상인가?

### 이미지
- [ ] 고화질 이미지인가? (800x600px 이상)
- [ ] 본문과 관련이 있는가?
- [ ] Alt 텍스트를 설정했는가?

### 기타
- [ ] 맞춤법과 띄어쓰기가 정확한가?
- [ ] 관련 다른 글 링크가 포함되었는가?
- [ ] 키워드가 자연스럽게 분포되어 있는가?
- [ ] 스팸성 표현은 없는가?

## 10. 최종 발행 팁

1. **발행 시간**: 오전 10-11시, 저녁 7-8시 (높은 접속률 시간)
2. **홍보**: 발행 후 SNS 공유 (카카오, 페이스북 등)
3. **후속 관리**: 1주일 후 댓글 답변, 2주일 후 추천수 확인
4. **정기 업데이트**: 월 2회 이상 꾸준히 발행
5. **연관글 클러스터**: 같은 주제의 글 여러 개로 검색 점유율 확대

---

## 참고사항

- 네이버 블로그 검색 알고리즘은 지속적으로 변경됨
- 사용자 경험(체류시간, 댓글)이 순위에 영향
- 신뢰도 높은 글(출처 명시, 전문성)이 유리
- 꾸준한 업로드가 가장 중요한 요소
